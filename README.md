# ASMR Client Portal Frontend

This is the frontend for the ASMR Client Portal prototype.  
It provides a clean, minimal client dashboard that interacts with the backend API.

---

## 🚀 Tech Stack
- **React 18** (UI)
- **Vite** (bundler)
- **Tailwind CSS** (styling)
- **React Router DOM** (routing)
- **React Icons** (icons)
- **React Toastify** (notifications)
- **html2canvas + jsPDF** (generate/download PDF reports)

---

## 📂 Features
- Login form with JWT authentication.
- Minimal dashboard displaying:
  - Client name
  - Project status (Draft, In Review, Approved)
- Button to download a sample PDF report.
- Toast notifications for success/error.
- Clean black/white UI with Tailwind.
